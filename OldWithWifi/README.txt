This OldWithWifi program is a now unused and unfinalised implementation of image processing and wifi serial connection strategies.
This involves use of the python OpenCV and Telnet communciations libraries to send location data to the Arduino as output from an overhead camera, followed by decision making on the side of the Arudino to acheive desired positions and orientations.
